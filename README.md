# Banking Management System

A web-based banking application built using Java, JSP, Servlets, JDBC, MySQL, and MVC Architecture. The application simulates core banking operations, enabling users to securely manage accounts, perform transactions, and track financial activities through an intuitive web interface.

## Features

* Secure User Registration & Login
* Account Dashboard with Balance Inquiry
* Deposit and Withdrawal Operations
* Fund Transfer Between Users
* Transaction History Management
* Session-Based Authentication
* JDBC Integration with MySQL
* MVC-Based Application Design

## Tech Stack

**Backend:** Java, Servlets, JDBC

**Frontend:** JSP, HTML, CSS

**Database:** MySQL

**Server:** Apache Tomcat 9

**Build Tool:** Maven

**Version Control:** Git & GitHub

## Project Highlights

* Implemented secure authentication and session management
* Developed fund transfer functionality with automatic balance updates
* Maintained transaction records for tracking and auditing
* Applied DAO and MVC design patterns for clean code organization
* Integrated MySQL database connectivity using JDBC
